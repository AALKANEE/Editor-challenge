# Challenge Editor

A simple command-line text editor built in **Go (Golang)** with basic and advanced features. This program allows you to add, delete, edit, and save text using straightforward commands.

## Features
- **Add Text**: Append new lines to the list.
- **Delete Text**: Remove lines by number or the last one by default.
- **List Content**: View the current content.
- **Edit Text**: Modify existing lines.
- **Save to File**: Export content to a file.
- **CLI Interface**: Simple and user-friendly.

## Prerequisites
- **Go**: Ensure you have Go (at least version 1.16) installed. Download it from [here](https://go.dev/dl/).
